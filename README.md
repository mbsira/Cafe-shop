# ☕ Cozy Cup Café

A modern café website built with React and Tailwind CSS.

🔗 **Live Demo:** [cafe-shop-k5o6.vercel.app](https://cafe-shop-k5o6.vercel.app/)

---

## Preview

![Hero](screenshots/cafe-home-page)
![Menu](screenshots/cafe-menu-page)

---

## Features
- 🛒 Add to cart & checkout
- 🔍 Filter menu by category
- 📱 Fully mobile responsive
- ✨ Smooth scroll reveal animations
- 🗺️ Location & contact section
