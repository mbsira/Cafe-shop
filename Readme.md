# ☕ Cozy Cup Café

A modern café website built with React and Tailwind CSS.

🔗 **Live Demo:** https://cafe-shop-k5o6.vercel.app/

## Features
- Browse menu by category
- Add items to cart
- Remove items and checkout
- Mobile responsive
- Scroll reveal animations

## Tech Stack
- React
- Vite
- Tailwind CSS (CDN)
- JavaScript
